# Logo

Coloque aqui os arquivos da logo do Clinix System.

Formatos recomendados:
- `logo.svg` — versão vetorial (melhor qualidade, qualquer tamanho)
- `logo.png` — versão PNG com fundo transparente (mínimo 400x400px)
- `logo-white.svg` — versão branca (para fundos escuros)
- `favicon.ico` ou `favicon.png` — ícone da aba do navegador (32x32px)
